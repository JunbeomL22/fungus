# Project Brief

## Project Name
Fungus

## Description
Fungus is a Java-based project that includes benchmarks using JMH (Java Microbenchmark Harness). The project appears to have a main application component and several benchmark implementations.

## Core Requirements
1. Main application functionality in `src/main/java/com/junbeom/App.java`
2. Benchmark implementations in `src/jmh/java/com/junbeom/`
3. Build and run scripts (mvn-compile.bat, run.bat)
4. Dependency management via Maven (pom.xml)
5. Git repository with remote origin

## Goals
1. Understand and document the project's purpose and functionality
2. Ensure benchmarks can be run and analyzed
3. Maintain and improve the main application
4. Document the system architecture and patterns
